Free Resources for Devs

Figma

Link: figma.com

Free collaborative interface design tool (great for devs too).

freeCodeCamp

Link: freecodecamp.org

Full free certifications in web dev, JS, Python, data science, and more.

Canva

Link: canva.com

Easy graphic design tool with thousands of free templates and elements.

Codewars

Link: codewars.com

Fun coding challenges (katas) to practice programming skills in many languages.

Inkscape

Link: inkscape.org

Free open-source vector graphics editor (alternative to Illustrator).

Flaticon

Link: flaticon.com

Huge library of free icons in SVG, PNG (attribution required for free version).

Pixabay

Link: pixabay.com

Free stock photos, illustrations, and vectors (no attribution needed).

W3Schools

Link: w3schools.com

Clear tutorials and references for HTML, CSS, JS, Python, SQL, and more.

Phosphor Icons

Link: phosphoricons.com

Beautiful, open-source icons (line, filled, duotone) MIT licensed.

Lucide Icons

Link: lucide.dev

Minimalist, customizable open-source icons (fork of Feather Icons).

Unsplash

Link: unsplash.com

High-quality free stock photos (no attribution required, great for websites/apps).

Pexels

Link: pexels.com

Free stock photos and videos (royalty-free for commercial use).

The Odin Project

Link: theodinproject.com

Full-stack web dev curriculum (HTML/CSS/JS/Node/React) 100% free.

MDN Web Docs

Link: developer.mozilla.org

Best free reference and learning docs for web technologies (HTML, CSS, JS).
